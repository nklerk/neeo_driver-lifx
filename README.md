# LIFX driver for NEEO

This driver adds support for LIFX to NEEO.

depending on the device, the following features are availeble:
- Power (switch)
- Light On (button)
- Light Off (button)
- Dimmer (Slider)
- Temperature (Slider)
- HUE (slider)
- Saturation (Slider)
- iR (Slider)
